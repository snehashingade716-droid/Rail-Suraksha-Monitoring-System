# 🚆 Rail Suraksha Smart Monitoring & Analytics System

## 📌 Project Overview
This project focuses on analyzing railway complaint data to enhance passenger safety and improve decision-making. It simulates a real-world monitoring system similar to railway safety applications.

## 🎯 Objectives
- Identify high-risk locations based on complaint data
- Analyze complaint types and patterns
- Monitor trends over time
- Provide insights for better safety management

## 🛠️ Technologies Used
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Power BI (Dashboard Visualization)

## 📊 Features
- Data cleaning and analysis using Python
- SQL database integration for structured data storage
- Interactive Power BI dashboard with filters (slicers)
- Visualization of complaint trends and patterns

## 📈 Key Insights
- Certain locations have higher complaint frequency
- Theft is the most common issue
- Complaints increase during night hours

## 🚀 Future Enhancements
- Real-time data integration
- GPS-based location tracking
- Alert system for high-risk zones
- Machine learning for prediction
